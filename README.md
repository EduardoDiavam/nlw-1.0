### Install API 

```bash
# Clone this repository
$ git clone https://github.com/EduardoDiavam/nlw-1.0.git

# Go into the repository
$ cd nlw-1.0/nlw-backend

# Install dependencies
$ yarn install

# Run Migrates
$ yarn knex:migrate

# Run Seeds
$ yarn knex:seed

# Start server
$ yarn dev

# running on port 3333
```

### Install Front-end
